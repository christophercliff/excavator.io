Chris here, creator of the Excavator project. Today I'm officially retiring excavator.io and decommissioning all services. The project has been on hiatus for the past several months and I simply don't have the resources to continue development. My sincere thanks to everyone who signed up.

Excavator was a Google Chrome™ extension and web application for collecting responsive screenshots. The project was started in December 2012 and retired in April 2014.

Several of the components are open source and MIT licensed:

  • dino | A DynamoDB object modeler for node.js
  • chrx | CommonJS modules for developing Chrome Extensions
  • fracas | Scrape web font meta data from the DOM
  • Excavator for Google Chrome™ | The Google Chrome™ extension
